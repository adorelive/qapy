查找Dockerfile并构建镜像x(当前目录) | docker build -t x .
列出本地镜像 | docker images
删除未使用的镜像x(强制删除) | docker rmi x -f
清理悬空镜像(包括未使用的镜像) | docker image prune -a
从仓库拉取镜像x的y版本 | docker pull x:y
为镜像x添加标签x:y | docker tag x x:y
导出镜像x为x.tar.gz文件 | docker save -o x.tar.gz x
从x.tar.gz文件导入镜像 | docker load -i x.tar.gz
导出容器x的快照为x.tar | docker export -o x.tar x
通过容器快照x.tar创建镜像x | docker import x.tar x
通过镜像x创建容器(自定义容器名称a) | docker run --name a x
通过镜像x创建容器(主机8080端口映射到容器80端口) | docker run -p 8080:80 x
通过镜像x创建容器(后台运行容器) | docker run -d x
通过镜像x创建容器(立即进入容器) | docker run -it x /bin/bash
通过镜像x创建容器(主机的/a挂载到容器的/b) | docker run -v /a:/b x
容器x的(启动)(停止)(重启) | docker start stop restart x
删除已停止的容器x(强制删除) | docker rm x -f
进入运行中容器x的Shell | docker exec -it x /bin/bash
查看容器x的日志(实时查看) | docker logs -f x
查看容器x的详细信息 | docker inspect x
列出正在运行的容器(包含已停止) | docker ps -a
查看容器x内的进程信息 | docker top x
查看所有容器的资源占用情况(包含未运行的) | docker stats -a
从容器x复制文件/a到宿主机的当前目录 | docker cp x:/a a
容器x的(暂停)(恢复) |  docker pause unpause x
查看容器x的端口映射 | docker port x
将容器x的当前状态保存为一个新的镜像s | docker commit x s
重命名容器x为s | docker rename x s
