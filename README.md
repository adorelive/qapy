# 📚 问答学习工具 (Quiz Learner)

一个基于命令行交互的轻量级问答学习工具，通过反复问答帮助记忆知识点。支持自定义题库、随机出题、自动提示答案，让学习变得高效又有趣。

---

## ✨ 功能特点

- 📂 **支持多题库切换** —— 从 `./notes/` 目录自动加载所有 `.txt` 格式的题库文件
- 🔀 **随机出题** —— 每次学习题目顺序随机打乱，避免顺序记忆
- 💡 **智能提示** —— 答错两次后自动显示答案，帮助加深印象
- 🎯 **即时反馈** —— 答对自动进入下一题，答错可继续尝试
- 🎨 **友好交互** —— 基于 `questionary` 构建美观的命令行界面

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install questionary
```

### 2. 准备题库

在项目根目录下创建 `notes` 文件夹，放入格式为 `问题 | 答案` 的 `.txt` 文件：

```
./notes/
├── 网络安全基础.txt
├── Python基础.txt
├── Linux命令.txt
└── ...
```

**题库格式示例**（`网络安全基础.txt`）：

```
OSI七层模型从下到上分别是哪七层？ | 物理层、数据链路层、网络层、传输层、会话层、表示层、应用层
ping命令基于什么协议？ | ICMP协议
HTTP和HTTPS的区别是什么？ | HTTPS比HTTP增加了加密功能，更安全
```

> ⚠️ **注意**：每行一条问答，问题与答案之间用 `|`（竖线）分隔，竖线前后可有空格。

### 3. 运行脚本

```bash
python3 qapy.py
```

|------|------|

## 🛠️ 核心代码解析

| 函数/代码段 | 功能说明 |
|-------------|----------|
| `os.listdir('./notes/')` | 扫描 `notes` 目录，获取所有题库文件名 |
| `questionary.select()` | 提供交互式选择菜单 |
| `line.strip().split('\|')` | 按 `\|` 分割题目和答案 |
| `random.shuffle(qa)` | 随机打乱题目顺序 |
| `while True` 循环 + `i` 计数器 | 允许最多尝试3次，第3次显示答案 |
